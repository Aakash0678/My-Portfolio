# Aakash Pandey | Software Engineer & AI Developer

[![Live Demo](https://my-portfolio-five-jade-i4qwaplymg.vercel.app/)

A high-performance, interactive 3D portfolio showcasing my work across Artificial Intelligence, Large Language Models (LLMs), and cloud-scalable software engineering. 

This repository contains the source code for my personal web space, designed to bridge the gap between foundational low-level performance (C++/DSA) and modern intelligent ecosystems (Python/GenAI).

## 🚀 Features

* **Interactive 3D Background**: Custom-built particle matrix using `Three.js` that reacts to mouse movement and gestures in real-time.
* **High-Fidelity Animations**: Smooth scroll triggers and element reveals powered by `GSAP` (GreenSock Animation Platform).
* **Modern UI/UX**: Fully responsive "Glassmorphism" design system utilizing `Tailwind CSS`.
* **Serverless Contact Integration**: Fully functional, secure contact form routed through `Formspree`.
* **Single-File Architecture**: Optimized for lightning-fast delivery and instantaneous rendering without heavy framework overhead.

## 🛠️ Tech Stack

* **Core**: HTML5, CSS3, Vanilla JavaScript
* **Styling**: Tailwind CSS (via CDN)
* **Animation & 3D**: Three.js (WebGL), GSAP + ScrollTrigger
* **Form Handling**: Formspree (Serverless POST endpoint)
* **Hosting**: Vercel / GitHub Pages
